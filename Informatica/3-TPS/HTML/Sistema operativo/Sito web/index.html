Cosè un Sistema Operativo

Il sistema opertivo è un software che serve a gestire il computer e interpreta i comandi che può dare l'utente o un software applicativo.

Premessa:

  1. Quando l'utente accende il computer viene eseguito un programma "bootstrap" che è memorizzato su un chip della scheda madre (memoria rom).
  2. Ogni componente hardware esegue un programma di diagnostica POST che controlla che tutto funzioni correttamente: CPU, scheda video, ram, tastiera...
  3. Viene caricato dall'hard disk una parte del sistema operativo sulla ram: kernel o nucleo del sistema operativo.

Compiti del SO

  1. Gestire le risorse hardware: CPU, RAM, ...
  2. Intergaccia tra l'utente e il computer: prende i comandi dall'utente e li fa eseguire all'hardware.
  3. Interfaccia tra il software applicativo e il computer.

Come è fatto:

É composto da vari programmi che parlano tra loro.

Il kernel è una parte del SO che è sempre presente sulla ram fino allo spegnimento del computer.


Struttura del SO:

  1. Nucleo (kernel)
  2. Gestore della memoria centrale
  3. Gestore pericferiche
  4. File system
  5. Interfaccia utente (shell)
  6. software applicativi

    7. Onion Skin (Buccia di cipolla)


I vari livelli vedono il livello sottostante come un oggetto a se (macchina virtuale) e dialoga con questo usando le primitive (procedure standard)

Il kernel:

É quella parte di sistema operativo che viene caricato subito in ram. É il livello che dialoga direttamente con l'hw.

I compiti del kernel sono:

  1. Avvio e chiusura dei programmi.
  2. Assegnazione della CPU ai vari processi.
  3. Sincronizzazione tra i processi.
  4. Sincronizzazione dei processi con l'ambiente esterno.

L'importanza del kernel

Il kernel isola completamente l'hw così tutto quello che funziona sopra è independente dall'hw e può funzionare su diversi computer (portabilità).


La Shell

La shell è l'interfaccia tra l'utente e il kernel. Le chiamate di funzione vengono fatte dalla shell sul kernel.

Può essere di 2 tipi:

CUI: Comand use interface.
GUI: Grafical use interface.
